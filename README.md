# Unturned-Auto-Fish
Allows for auto fishing on unturned using spikes in game audio, having a pretty high success rate, will most likely get you banned from most servers

download python
run a cmd in that folder, type cmd into the path section google this if need help
download the python extensions using this command:
pip install -r requirements.txt

download vb virtual cable (this captures the audio)
https://vb-audio.com/Cable/
download the free windows thing
open it up
set your windows audio to vb virtual cable in your windows settings

run the script itself using:
python autofish.py

the script:
fishes when a loud sound is heard, so reduce all other sounds to 0
let it run


DO NOT USE ON SERVERS WITHOUT PERMISSION
